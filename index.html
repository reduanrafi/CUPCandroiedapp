Contest

IUCPC

City university Programming club organizing a regular programming contest per semester Named: Intra University Computer Programing Contest(IUCPC). where Teams and individuals can take part.

Rules

There are few rules to take part in Each Programming Contests.

  • In Individual Contest every single contestant must complete registration process
  • In team wise contest Team leader must complete registration process providing other members Detail
  • Each team can have 3 members at most

    A program to Print a string

    #include
    int main()
    {
    /* My first program */
    printf("I am a programmer \n");
    return 0;
    }

    Description:
    Here it’s the structure of a Basic C program. This program will out put a line “I am a programmer”. Why we write all of those lines to print a single sentence .you will get all of this information in the previous chapter .

    Now, Here # is called the preprocessor #include means we included standard input output header file.

    main() is a function called the brain function of a program .without this main function no program will execute. A program may have many functions or not but a main() is a must to execute program.

    /* My first program */ this a comment line it have no effect on program execution .

    Printf(“”) it’s a library function stored in the header file we described previously ; is used to print something in a program.

    “” inside this quotation we may write whatever we want..

    \n is used to print a new line .

    {}this are called curly bracket must needed to write a function including main function. “{“start a function and “ } “ends the function.

    The final and most important thing is ;(semicolon )it is used at the end of each line.

    Note: int and return 0 will discussed latterly.

    practice

      2.write a program to write your name and your address in two separate line using a single printf() function.
      3.write a program to print your educational qualification. (use as much printf functions as you need. )

    A program to Print a number and discuss about data type and variable

    #include
    int main()
    { /* My second program */
    int a =1;
    printf("The value of a=%d\n",a);
    return 0;
    }

    Description:
    This our second program derived from first one . here we just included one extra line
    DATA TYPE:
    Int a =10;
    Here int is a data type
    In programming we used several types of data.
    C has four basic data type
    1.int is used for integer type number like 1, 2,3 etc.
    2.float for floating point number like 1.2 ,2.5, etc.
    3.char is used for character like a,b,c.. etc .
    4.double same as floating point but its range is more then float.
    In the printf function we use %d its called identifier. Used to print integer type data .
    %f for floating point number
    %c character and
    %lf for double .
    Variable: In the new line int a =1; a is called variable where we keep the valu.
    Here 1 is kept in a .

    practice

      1.write a program to print a floating point number
      2.write a program to print a floating and a an integer number in different line using a printf function.
      3. write a program to print a character .

    A program to use Operator

    #include
    int main()
    { /* My third program */
    int a =1,b=2,c=0;
    c=a+b;
    printf("The value of c=%d\n",c);
    return 0;
    }

    Description:
    Operator :
    Here we have 3 variables a b and c.
    All are integer type
    Here we use a ’+’ sign and this is called operator.
    There are Few types of operator
    1 arithmetic operator(+,-,*,/)
    2 logical operator. (||,&& etc)
    3.Relational Operator.(<,>,<= etc)
    4.Assignment operator(==,++,--) This program will print the value of c=3;
